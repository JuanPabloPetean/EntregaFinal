Sobre Este Proyecto

Este proyecto es una aplicación web desarrollada con Django para gestionar obras de teatro. Permite a los usuarios crear, visualizar, editar y eliminar obras, con un sistema completo de autenticación de usuarios.

Funcionalidades Principales

  • Sistema de Autenticación: Login, logout y registro de usuarios
  • Gestión de Obras: CRUD completo para obras de teatro
  • Interfaz Intuitiva: Diseño responsivo y fácil de usar
  • Imágenes: Soporte para cargar imágenes de obras
  • Admin Panel: Gestión desde el panel administrativo de Django

Tecnologías Utilizadas

  • Django 5.2.8
  • Python 3.13
  • SQLite3
  • HTML/CSS
